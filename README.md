# telegram_bot
Project for MIPT net course

## Dependencies

* Ubuntu:
    * cmake `sudo apt-get install cmake`
    * gcc TODO
    * Poco `sudo apt-get install libpoco-dev`
* MacOS:
    * cmake `brew install cmake`
    * gcc `brew install gcc`
    * Poco `brew install poco --build-from-source --cc=gcc-8`
